Remarks

Implementations must be thread-safe.

Many Terraria objects are entities and have common properties such as position, velocity, etc. Entities are annotatable, allowing consumers to easily attach custom state to them.

There are three types of entities:

  1. Entities which are abstract: they cannot affect the world.
  2. Entities which are concrete but not active: the entity does not exist in the world.
  3. Entities which are concrete and active: the entity exists in the world.

These types can be differentiated using the IsConcrete(IEntity) extension method and the IsActive property.
