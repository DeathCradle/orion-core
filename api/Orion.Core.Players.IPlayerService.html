Interface IPlayerService

Represents a player service. Provides access to players and publishes packet and player-related events.

Namespace: Orion.Core.Players
Assembly: Orion.Core.dll
Syntax
[Service(ServiceScope.Singleton)]
public interface IPlayerService
Remarks

Implementations are required to be thread-safe.

The player service is responsible for publishing the following packet and player-related events:

  • PacketReceiveEvent<TPacket>
  • PacketSendEvent<TPacket>
  • PlayerTickEvent
  • PlayerQuitEvent
  • PlayerJoinEvent
  • PlayerHealthEvent
  • PlayerPvpEvent
  • PlayerPasswordEvent
  • PlayerManaEvent
  • PlayerTeamEvent
  • PlayerUuidEvent
  • PlayerChatEvent
