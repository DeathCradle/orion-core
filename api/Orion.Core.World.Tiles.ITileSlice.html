Interface ITileSlice

Represents a two-dimensional slice of tiles.

Namespace: Orion.Core.World.Tiles
Assembly: Orion.Core.dll
Syntax
public interface ITileSlice
Remarks

Implementations are required to be thread-safe.

A tile slice can be used to efficiently pass tiles by reference in, e.g., aTileSquarePacket instance or the SendTiles(IPlayer, Int32, Int32, ITileSlice) extension method.

The Slice(ITileSlice, Int32, Int32, Int32, Int32) extension method allows zero-copy subslices to be obtained.
