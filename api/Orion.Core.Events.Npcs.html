Namespace Orion.Core.Events.Npcs

Classes

NpcBuffEvent

An event that occurs when an NPC is being buffed. This event can be canceled.

NpcCatchEvent

An event that occurs when a player is catching an NPC. This event can be canceled.

NpcDefaultsEvent

An event that occurs when an NPC's defaults are being set: i.e., when an NPC's stats are being initialized. This event can be canceled.

NpcEvent

Provides the base class for an NPC-related event.

NpcFishEvent

An event that occurs when a player is fishing an NPC. This event can be canceled.

NpcKilledEvent

An event that occurs when an NPC has been killed.

NpcLootEvent

An event that occurs when an NPC is dropping loot. This event can be canceled.

NpcSpawnEvent

An event that occurs when an NPC is spawning. This event can be canceled.

NpcTickEvent

An event that occurs when an NPC tick (update) occurs. This event can be canceled.
